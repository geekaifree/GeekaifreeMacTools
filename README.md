## 🚀 快速开始

1. 下载 `.dmg` 文件
2. 双击打开，拖拽 `.app` 到 Applications 文件夹
3. 启动使用！

> 💡 **提示**：所有应用均为 Universal Binary，同时支持 Apple Silicon (M1/M2/M3) 和 Intel 芯片的 Mac。
